    常见居中总结

    vasonYin 2021-3-12 CSS

    常见元素居中分析与总结。

    HTML 结构

    <div class="container">
  <div class="center">水平居中元素</div>
</div>

    # 水平居中

    # 内联元素

    对 inline、inline-block、inline-table 和 inline-flex 元素水平居中都有效。

    <!-- HTML -->
<div class="container">
  <span class="center">水平居中元素</span>
</div>
<!-- css -->
<style>
  .container {
    text-align: center;
  }
</style>

    # 块级元素

    # margin: auto

    必须是定宽元素

    .center {
  width: 200px;
  margin: 0 auto;
}

    # table + margin: auto

    display:table在表现上类似 block 元素，子元素宽度为内容宽。

    .center {
  display: table;
  margin: 0 auto;
}

    # absolute+负 margin

    必须是定宽元素

    .center {
  position: absolute;
  left: 50%;
  margin-left: -100px;
  width: 200px;
}
.container {
  position: relative;
}

    # absolute+transform

    .center {
  position: absolute;
  left: 50%;
  transform: translateX(-50%);
}
.container {
  position: relative;
}

    # 多个元素

    # inline-block

    行内块存在默认间距问题，需要清除 inline-block 默认间距

    .container {
  text-align: center;
}
.center {
  display: inline-block;
}

    # 浮动元素

    # relative+负 margin

    必须是定宽元素

    .center {
  float: left;
  position: relative;
  left: 50%;
  margin-left: -100px;
  width: 200px;
}

    # relative+relative

    要清除浮动，给外部元素加上 float 父元素与子元素等宽等高

    .container {
  float: left;
  position: relative;
  left: 50%;
}

.center {
  float: left;
  position: relative;
  right: 50%; //left: -50%;
}

    # 绝对定位元素

    # absolute+margin: auto（推荐）

    .container {
  position: relative;
}
.center {
  position: absolute; /*绝对定位*/
  width: 200px;
  height: 100px;
  margin: 0 auto; /*水平居中*/
  left: 0; /*此处不能省略，且为0*/
  right: 0; /*此处不能省略，且为0*/
}

    # flex

    通用方法，不需知道子元素是定宽还是不定宽，是一个还是多个。

    .container {
  display: flex;
  justify-content: center;
}

    # 垂直居中

    # 单行内联元素

    设置行高等于元素高度

    <!-- HTML -->
<div class="container">
  <span class="center">垂直居中元素</span>
</div>
<!-- css -->
<style>
  .container {
    height: 40px;
    line-height: 40px;
  }
</style>

    # 多行内联元素

    # flex

    <div class="container">
  <p>
    Dance like nobody is watching, code like everybody is. Dance like nobody is
    watching, code like everybody is. Dance like nobody is watching, code like
    everybody is.
  </p>
</div>
<style>
  .container {
    height: 140px;
    display: flex;
    flex-direction: column;
    justify-content: center;
  }
</style>

    # table

    子元素与父元素等宽

    <div class="container">
  <p class="center">
    The more technology you learn, the more you realize how little you know. The
    more technology you learn, the more you realize how little you know. The
    more technology you learn, the more you realize how little you know.
  </p>
</div>
<style>
  .container {
    display: table;
    height: 140px;
  }
  .center {
    display: table-cell;
    vertical-align: middle;
  }
</style>

    # 块级元素

    # absolute+margin

    兼容性不错,但是必须是定高元素

    .container {
  position: relative;
}
.center {
  position: absolute;
  top: 50%;
  height: 100px;
  margin-top: -50px;
}

    # absolute+transform

    .container {
  position: relative;
}
.center {
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

    # table-cell

    同样适用于多行文字的垂直居中处理

    .container {
    display: table-cell;
    vertical-align: middle;
}

    或者

    .container {
  display: table;
}
.center {
  display: table-cell;
  vertical-align: middle;
}

    # flex

    .container {
    display:flex;
    align-items:center;
}

    # 水平垂直居中

    # table

    .container {
  display: table-cell;
  vertical-align: middle;
  text-align: center;
}
.center {
  display: inline-block; /* 适合多列 */
}

    # absolute+负 margin

    margin 平移元素整体宽度的一半

    .container {
  position: relative;
}

.center {
  position: absolute;
  top: 50%;
  left: 50%;
  margin: -50px 0 0 -50px;
  width: 100px;
  height: 100px;
}

    # absolute+margin：auto

    不需要提前知道尺寸，兼容性好

    .container {
  position: relative;
}

.center {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  margin: auto;
}

    # absolute+transform

    利用 2D 变换，在水平和垂直两个方向都向反向平移宽高的一半

    .container {
  position: relative;
}
.center {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%); }

    # flex

    利用 flex 布局，其中justify-content 用于设置或检索弹性盒子元素在主轴（横轴）方向上的对齐方式；而align-items属性定义 flex 子项在 flex 容器的当前行的侧轴（纵轴）方向上的对齐方式。

    .container {
  height: 100vh; /* 必须有高度 */
  display: flex;
  justify-content: center;
  align-items: center;
}

    # flex/grid+margin:auto

    利用 grid 实现水平垂直居中

    .container {
  height: 100vh;//必须有高度
  display: grid;
}
.center {
  margin: auto;
}

    # 屏幕上水平垂直居中

    场景：元素相对屏幕水平垂直居中，文本超出宽度换行显示

    HTML

    <div class="outer">
  <div class="middle">
    <div class="inner"></div>
  </div>
</div>

    CSS

    .outer {
  display: table;
  position: absolute;
  height: 100%;
  width: 100%;
}

.middle {
  display: table-cell;
  vertical-align: middle;
}

.inner {
  margin-left: auto;
  margin-right: auto;
  width: 400px;
}
